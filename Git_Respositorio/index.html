• git init Cria um repositório git dentro da pasta
  • git status Estatus do repositório
  • git add nome Adiciona o arquivo no stage
  • git add . Adiciona todos os arquivos no stage
  • git commit -m "msg" Msg que vai ser enviada para o repositório remoto
  • git remote add origin + end do github Vincula seu repositório local com o remoto
  • git log Mostra todos os commit feitos
  • git push origin master Vincula o repositório local ao remoto
  • git clone end do repo. Clona o repositório do github
  • git diff Mostra o que foi modificado (add ou remo) no arquivo
  • git pullVerifica se tem alguma atualização pendente
  • git checkout -b nome Cria uma nova branch
  • git checkout nome Troca de branch
  • git merge nome Adiciona todas as modificações na branch master
  • git branch -D nome Deleta a branch
