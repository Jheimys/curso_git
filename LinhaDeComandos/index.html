Lista de comandos do git

  • pwd: Mostra o diretório que estamos
  • ls: Lista os arquivos dentro do diretório
  • ls -a: Lista tudo que temos dentro do diretório
  • ls --help: Exibe a documentação do comando
  • clear Para limpar a tela, o atalho é Ctrl + L
  • Ctrl + C: Interrompe o aquirvo ue esta sendo execultado
  • Seta para cima: Mostra os ultimos comandos que foram execultados
  • mkdir + nome: Cria diretórios
  • rmdir + nome: Remove diretórios vazios
  • rm arquivo: Remove arquivo
  • rm -rf: Remove diretórios que possuem arquivos
  • cd: Entra no diretório
  • cd .. : Volta para diretório anterior
  • touch +nome: Cria arquivos
  • mv Arquivo Pasta : move um arquivo para pasta específicada
  • code . : Abre o vscode
